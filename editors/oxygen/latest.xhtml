This add-on installs a recent snapshot of the development version of XSpec. See Wiki for details.

This add-on does not work side by side with XSpec Framework add-on (a companion add-on for XSpec Helper view add-on).

Changelog (not inclusive)

v3.2.1

  • Release Candidate of the stable release
  • Code coverage report includes Contents section with coverage statistics (#2078).
  • Tested with BaseX 11.7.

v3.2.0

  • Test reports use black text on white background by default, improving accessibility. Themes 'whiteblack' (white on black) and 'classic' (earlier green/pink design) are also available. (#2055)
  • Tests for Schematron can verify string values of Schematron properties (#2045).
  • Tested with BaseX 11.6
  • Tested with Oxygen 27.0.

v3.1.3
